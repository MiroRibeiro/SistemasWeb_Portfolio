Conhecimentos

HTML

O HTML5 é a nova versão do HTML e um dos seus principais objetivos é facilitar a manipulação dos elementos fazendo com que isso fique transparente para o usuário final.

Css

CSS é a sigla para Cascading Style Sheets, que significa Folha de Estilo em Cascatas. É uma linguagem de formatação de conteúdo que permite estilizar elementos escritos em HTML ou XHTML.

Javascript

JavaScript é uma linguagem de programação interpretada, baseada em objetos e de alto nível, usada em sites, aplicativos e sistemas online.

ReactJS

O ReactJS é uma biblioteca front-end JavaScript de código aberto com foco em criar interfaces de usuário em páginas web.

Git e Github

GitHub é uma plataforma de hospedagem de código-fonte usando o Git, permitindo que programadores contribuam em projetos de qualquer lugar do mundo.

Python

Python é uma linguagem de programação de alto nível, interpretada, imperativa, oo, funcional, de tipagem dinâmica e forte.
